## Project Overview

This project demonstrates network traffic analysis using Wireshark in a virtual lab environment. Network packets were captured and inspected to understand communication patterns, analyze protocols, and identify suspicious network activity.

---

## Objectives

- Capture live network traffic
- Analyze DNS traffic
- Inspect HTTP requests and responses
- Understand TCP communication
- Monitor ICMP packets
- Perform packet inspection
- Investigate suspicious network behavior
